# Week 0 — Weather Project (Scratch)

## Overview
This repository contains my **Harvard CS50x 2026 – Week 0** Scratch project.  
The project is a small interactive scene where a penguin character can interact with different clothing items, inspired by a simple weather/clothing concept.

---

## Play
You can play the project directly on Scratch:

https://scratch.mit.edu/projects/1262565596/

> Click the **green flag** to start the project.

---

## Repository Structure
- `week0/week0-weather-project.sb3` — The Scratch project file (working version).
- `week0/README.md` — This file, containing the project description and play link.
- `README.md` — Main repository description.

> Note: The `.sb3` file can be opened in the Scratch editor or downloaded.  
> This is the same file format used for CS50 submission.

---

## How to Play
1. Open the Scratch project link above.  
2. Click the **green flag (▶)** to start.  
3. Click on different clothing items to change the penguin’s appearance.
